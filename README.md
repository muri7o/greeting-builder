# Greeting Builder

## 📌 Sobre o projeto
Este projeto é um exercício de JavaScript que gera uma mensagem de saudação com base no nome da pessoa e no período do dia.

Link do desafio: https://roadmap.sh/projects/js-greeting-builder

## ⚙️ Funcionalidades
- Formata o nome completo
- Retorna saudação baseada no período do dia (manhã, tarde, noite)
- Combina tudo em uma mensagem final

## ▶️ Como executar

Certifique-se de ter o Node.js instalado.

Execute o arquivo com:

```
node greeting.js
```

## 🧪 Exemplo de saída

```
Good morning, Murilo Oliveira!
```
